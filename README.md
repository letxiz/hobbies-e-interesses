# 🎨 Hobbies e Interesses — HTML & CSS

Projeto de estudo em desenvolvimento front-end utilizando **HTML5** e **CSS3**.  
O site apresenta meus hobbies e interesses com layout moderno e organização semântica.

---

## 📚 Conceitos aplicados

- Estruturação semântica com HTML5  
- Organização de seções (`header`, `nav`, `main`, `section`, `footer`)  
- Layout com Flexbox  
- Responsividade com Media Queries  
- Separação entre HTML e CSS  
- Estilização moderna  
- Organização de arquivos  

---

## 🧱 Estrutura do Projeto

📄 `index.html`  
Arquivo principal contendo toda a estrutura do site.

🎨 `styless.css`  
Responsável pela estilização e layout visual.

🖼 `assets/`  
Pasta contendo as imagens utilizadas nas seções.

---

## 🖥 Seções do Site

- 🏠 Home (Boas-vindas)
- 🎮 Games
- 🎥 Lives
- 🎬 Filmes
- 🎵 Músicas
- 🔗 Links (com playlist incorporada do Spotify)

---

## ▶️ Funcionalidades

- Navegação por âncoras
- Layout alternado com imagem e texto
- Estrutura responsiva para dispositivos móveis
- Integração com playlist do Spotify
- Links externos para Twitch

---

## 🎯 Objetivo

Praticar conceitos fundamentais de desenvolvimento front-end, organização de layout e estruturação de páginas responsivas como base para projetos maiores.
